Project Overview

A web-based restaurant ordering and management system designed to simplify customer ordering, order tracking, payment monitoring and sales reporting. The system allows customers to place orders directly from their smartphones while helping restaurant owners manage daily operations more efficiently.

Problem Solved

Many small restaurants still rely on manual order taking and handwritten records. This system helps:

  • Reduce order-taking errors
  • Improve order tracking
  • Simplify payment monitoring
  • Organize sales records
  • Improve operational efficiency
Key Features
QR Ordering System

Customers scan a QR code and enter their table number before placing orders.

Order Management Dashboard

Restaurant staff can view and manage incoming orders in real time. Features include active orders, order status tracking, and payment status monitoring.

Sales Dashboard

Provides business insights including daily sales, historical sales records, order statistics, and revenue tracking.

Menu Management

Restaurant owners can add menu items, edit menu information, update pricing, and manage menu availability dynamically.

PDF Report Generation

Allows restaurant owners to download and keep detailed order records for future reference.

Technologies Used
Google Gemini Firebase Supabase

Project Overview

A modern digital wedding invitation platform designed to improve guest engagement and provide wedding information in a convenient and interactive format.

Problem Solved

Traditional wedding invitations provide limited interaction and often require guests to search separately for event details and directions. This platform provides all important information in one place.

Key Features
Real-Time Guest Wishes

Guests can submit wishes and messages that appear instantly without requiring page refresh.

Event Information Hub

Provides important wedding information including event details, venue information, and guest instructions.

Wedding Itinerary

Displays the event schedule to help guests understand the flow of the ceremony and reception.

Location & Navigation

Integrated location guidance helps guests easily find the wedding venue.

Background Music Experience

Creates a more engaging and personalized invitation experience.

Mobile-Friendly Design

Optimized for smartphones and tablets for easy access and sharing.

Technologies Used
Google Gemini Firebase
